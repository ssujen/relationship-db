# Relationship Knowledge Base Backend

This backend provides a gRPC service for managing and querying a personal relationship knowledge base. This project was generated by Antigravity based on relationship-database.txt.

## Features

- **Add Entity**: Store persons or objects with custom properties.
- **Add Relationship**: Define directed, typed relationships between entities (e.g., "husband_of", "hates").
- **Query Relationships**: Search for connections using source, target, or relationship type.

## Project Structure

- `src/server.ts`: gRPC server implementation.
- `src/database.ts`: SQLite database layer with graph-oriented schema.
- `src/client.ts`: Mock client for verification.
- `proto/relationship.proto`: gRPC service definition.

## Setup & Running

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Start the server**:
   ```bash
   npx ts-node src/server.ts
   ```
3. **Run the test client**:
   ```bash
   npx ts-node src/client.ts
   ```

## Database

The data is stored in a SQLite file at `data/relationships.db`.

## Client Integration

Android or other clients should use the `proto/relationship.proto` file to generate client stubs. We recommend using Gemini Nano on the client to parse natural language into the structured gRPC calls.
